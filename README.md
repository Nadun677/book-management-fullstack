# Book Management System

A simple full-stack web application built with Angular and ASP.NET Web API that allows users to manage books.

Language: C#

Features:
- View Books
- Add Book
- Edit Book
- Delete Book

## How to Run the Backend

Open a terminal and run the following commands:

cd BookApi  
dotnet run

The backend API will start on:
http://localhost:5000


## How to Run the Frontend

Open another terminal and run the following commands:

cd book-frontend  
ng serve

Then open your browser and go to:
http://localhost:4200
